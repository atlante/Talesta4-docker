Doc sur les différents types d'objets, leur créations, et comment en rajouter

 

1. Les différents types d'objets apportés par la 3.4
  1. Les Produits Naturels : Ce sont des objets que l'on peut "récolter" en se promenant dans la nature (Ex: Champignon dans une foret, Minerai de Fer dans une mine, Bois, Cuir, Pierre dans une carrière ...) Ces objets sont à affecter dans un magasin de produits naturels pour que les PJs puissent les récolter. Ces magasins gerent une quantite et une periode de regeneration ou non (Cette notion pourrait etre etendu aux autres magasins dans une version ultérieure)
    Pour en ajouter, il suffit d'en ajouter dans include/const.php dans $liste_type_objs avec les autres 'ProduitNaturel', puis de créer un objet de ce type avec l'application.
  2. Les Objets Simples : Ce sont des objets d'un seul matériau (Manche en bois, tete de hache ..) qui servent a composer des objets 'finaux'.
    Leur sous-type est le matériau en question. Pour le moment, le reste des champs (durabilite....) n'a pas d'importance (Ce sont les champs des objets finaux qui en ont)
    Pour en ajouter, il faut les créer dans l'application sans oublier de mettre le matériau en sous-type
  3. Les Outils : Ce sont des objets utilisables par les PJs pour
    • "récolter" les produits naturels,
    • fabriquer des objets simples à partir de ces produits naturels,
    • ou pour réparer des produits finaux

    Leur sous-type est la compétence pour laquelle ils sont utilisés. Le champ compétence est identique. Le champ caractéristique sert (en plus de la compétence) pour la réussite de l'action
2. Les différentes actions de l'artisanat apportés par la 3.4
  1. Recolter. Cette action permet de récolter un Produit Naturel fournit par le lieu. Suivant le type du Produit, cela nécessite un outil ou non. Ex: Recolter du minerai necessite un objet de type 'outil', et de sous type 'Mineur', ramasser un champignon ne necéssite pas d'outil.
  2. Artisanat. Cette action permet de créer des objets simples (d'un seul matériau) à partir de produit naturels de ce matériau Cette action necessite un objet de type 'outil', et de sous type de la compétence artisanale associée à l'action.
  3. Combiner objets. Cette action permet de créer des objets 'finaux' à partir d'objets simples (d'un seul matériau) La compétence utilisée est le dernier parametre de $liste_type_objs dans include/const.php (ArtisanArmeMelee pour une arme de Melee, ArtisanOutil pour un outil ...)
  4. Reparer : Cette action permet de réparer un objet abimé. Pour le moment, les types d'objets réparables sont:
    • ArmeJet.
    • ArmeMelee
    • Armure
    • Outils
    La compétence utilisée est le dernier parametre de $liste_type_objs dans include/const.php (ArtisanArmeMelee pour une arme de Melee, ArtisanOutil pour un outil ...)

4. Comment personaliser, ajouter, modifier ?
  1. Ajouter des types d'objets à réparer:
    • Modifier reparation.php pour ajouter une action qui determine les objets à réparer ($ObjetsReparables) et le sous-type d'outil utilisable ($sousTypeOutil)
    • Modifier menu_jeu.php pour ajouter l'action dans le menu
    • Modifier const.php pour ajouter une ligne dans liste_type_objs du type "Outil;ArtisanXXXXX"=>array("Outil - ArtisanXXXXX","Bras",2,1,"ArtisanOutil"), avec XXXXX le type d'objets à réparer
    • bien sur, creer l'outil avec le sous-type ArtisanXXXXX
    • Modifier const.php pour ajouter une ligne dans liste_artisanat du type ArtisanXXXXX
  1. Ajouter des types d'objets à récolter:
    • Modifier recolte.php pour ajouter une action qui determine les objets à récolter ($SousTypeProduit), l'obligation d'avoir un outil ou non ($necessiteOutil) pour l'action et le sous-type d'outil utilisable ($sousTypeOutil)
    • Modifier menu_jeu.php pour ajouter l'action dans le menu
    • Modifier const.php pour ajouter une ligne dans liste_type_objs du type "Outil;XXXXXX"=>array("Outil - XXXXXXX","Bras",2,1,"ArtisanOutil"), avec XXXXX le sous-type de l'outil
    • bien sur, creer l'outil avec le sous-type XXXXX
    • Modifier const.php pour ajouter une ligne dans liste_artisanat du type $SousTypeOutil
