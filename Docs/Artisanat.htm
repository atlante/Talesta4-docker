Doc sur les diffrents types d'objets, leur crations, et comment en rajouter

 

1. Les diffrents types d'objets apports par la 3.4
  1. Les Produits Naturels : Ce sont des objets que l'on peut "rcolter" en se promenant dans la nature (Ex: Champignon dans une foret, Minerai de Fer dans une mine, Bois, Cuir, Pierre dans une carrire ...) Ces objets sont affecter dans un magasin de produits naturels pour que les PJs puissent les rcolter. Ces magasins gerent une quantite et une periode de regeneration ou non (Cette notion pourrait etre etendu aux autres magasins dans une version ultrieure)
    Pour en ajouter, il suffit d'en ajouter dans include/const.php dans $liste_type_objs avec les autres 'ProduitNaturel', puis de crer un objet de ce type avec l'application.
  2. Les Objets Simples : Ce sont des objets d'un seul matriau (Manche en bois, tete de hache ..) qui servent a composer des objets 'finaux'.
    Leur sous-type est le matriau en question. Pour le moment, le reste des champs (durabilite....) n'a pas d'importance (Ce sont les champs des objets finaux qui en ont)
    Pour en ajouter, il faut les crer dans l'application sans oublier de mettre le matriau en sous-type
  3. Les Outils : Ce sont des objets utilisables par les PJs pour
    • "rcolter" les produits naturels,
    • fabriquer des objets simples partir de ces produits naturels,
    • ou pour rparer des produits finaux

    Leur sous-type est la comptence pour laquelle ils sont utiliss. Le champ comptence est identique. Le champ caractristique sert (en plus de la comptence) pour la russite de l'action
2. Les diffrentes actions de l'artisanat apports par la 3.4
  1. Recolter. Cette action permet de rcolter un Produit Naturel fournit par le lieu. Suivant le type du Produit, cela ncessite un outil ou non. Ex: Recolter du minerai necessite un objet de type 'outil', et de sous type 'Mineur', ramasser un champignon ne necssite pas d'outil.
  2. Artisanat. Cette action permet de crer des objets simples (d'un seul matriau) partir de produit naturels de ce matriau Cette action necessite un objet de type 'outil', et de sous type de la comptence artisanale associe l'action.
  3. Combiner objets. Cette action permet de crer des objets 'finaux' partir d'objets simples (d'un seul matriau) La comptence utilise est le dernier parametre de $liste_type_objs dans include/const.php (ArtisanArmeMelee pour une arme de Melee, ArtisanOutil pour un outil ...)
  4. Reparer : Cette action permet de rparer un objet abim. Pour le moment, les types d'objets rparables sont:
    • ArmeJet.
    • ArmeMelee
    • Armure
    • Outils
    La comptence utilise est le dernier parametre de $liste_type_objs dans include/const.php (ArtisanArmeMelee pour une arme de Melee, ArtisanOutil pour un outil ...)

4. Comment personaliser, ajouter, modifier ?
  1. Ajouter des types d'objets rparer:
    • Modifier reparation.php pour ajouter une action qui determine les objets rparer ($ObjetsReparables) et le sous-type d'outil utilisable ($sousTypeOutil)
    • Modifier menu_jeu.php pour ajouter l'action dans le menu
    • Modifier const.php pour ajouter une ligne dans liste_type_objs du type "Outil;ArtisanXXXXX"=>array("Outil - ArtisanXXXXX","Bras",2,1,"ArtisanOutil"), avec XXXXX le type d'objets rparer
    • bien sur, creer l'outil avec le sous-type ArtisanXXXXX
    • Modifier const.php pour ajouter une ligne dans liste_artisanat du type ArtisanXXXXX
  1. Ajouter des types d'objets rcolter:
    • Modifier recolte.php pour ajouter une action qui determine les objets rcolter ($SousTypeProduit), l'obligation d'avoir un outil ou non ($necessiteOutil) pour l'action et le sous-type d'outil utilisable ($sousTypeOutil)
    • Modifier menu_jeu.php pour ajouter l'action dans le menu
    • Modifier const.php pour ajouter une ligne dans liste_type_objs du type "Outil;XXXXXX"=>array("Outil - XXXXXXX","Bras",2,1,"ArtisanOutil"), avec XXXXX le sous-type de l'outil
    • bien sur, creer l'outil avec le sous-type XXXXX
    • Modifier const.php pour ajouter une ligne dans liste_artisanat du type $SousTypeOutil
