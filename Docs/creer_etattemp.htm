RPO : RPA : RPI : RPV :
RPO represente le nombre de Pieces d'OR que le PJ avec cet Etat gagnera a chaque remise de PAs. Peut etre négatif
RPA represente le nombre de Points d'action qu'un PJ peut reporter d'un tour a l'autre. Ne peut qu'etre positif ou nul.
RPI represente le nombre de Points d'Intellect qu'un PJ peut reporter d'un tour a l'autre. Ne peut qu'etre positif ou nul.
RPV represente le nombre de Points de Vie que le PJ gagnera a chaque remise. Peut etre négatif. Peut entrainer la mort.
