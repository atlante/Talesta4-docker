Pre-Requis: Le moteur Talesta4 est crit en php. Son utilisation ncessite un serveur HTTP et une base de donnes. Pour le moment, les seules bases de donnes utilisables sont Mysql (car elle est la base la plus souvent offerte par les hbergeurs), PostGreSQL qui est une base open-source avec plus de possibilits que Mysql, et Oracle (car elle est ma base prfre, je dconseille aux dbutants d'utiliser Oracle (beaucoup plus long installer, configurer ...)) Les moyens les plus simples pour se procurer Mysql et un serveur HTTP sont:
  • wampserver (http://www.wampserver.com/). La version utilise pour ce document est WAMP5.1.4.4
  • OU easyphp (http://www.easyphp.org). La version utilise pour ce document est Easyphp 1.7
Le moyen le plus simple pour se procurer PostGreSQL sous windows est: http://pgfoundry.org/projects/pginstaller La version utilise pour ce document est 8.0 Beta 5
Les versions ultrieures de tous ces systmes devraient fonctionner de la mme faon.

 

Notes techniques: Compatibilits avec les diffrentes versions de PHP: Cette version de Talesta4 est compatible avec toutes versions de PHP 4 suprieures ou gales la 4.3.0 ou 5 Compatibilits avec les diffrentes versions de Mysql: Toute version de Mysql 4 et 5 et de nombreuses versions Mysql 3 sont utilisables. A noter certaines requetes ont ete optimises pour les possesseurs de Mysql > 4.1 qui permet enfin les sous-requetes... Compatibilits avec les diffrentes versions d'Apache: Cette version de Talesta4 est compatible (a priori, je n'ai travaill qu'avec Apache 1.3.27 et 2.0.48) avec toutes versions d'Apache 1.3 ou 2.0

 

1. Installer EasyPHP
  1. Rcuprer Easyphp sur http://www.easyphp.org
  2. Lancer easyphp1-7_setup.exe pour installer le tout dans le rpertoire de votre choix (dnomm par la suite dans ce document [RepEasyPHP])
  3. Une fois l'installation termine, vous avez un client Web (Internet Explorer ou Netscape ou ...) ouvert avec la page d'accueil d'easyPHP affiche ([RepEasyPHP]\home\accueil.html).
  4. Lancer EasyPHP (dmarrer, programmes, EasyPHP 1.7, EasyPHP). Une icne (un E) apparait dans la barre de tches et aprs quelques secondesn un petit carr rouge clignote, signifiant qu'EasyPHP est lanc).
  5. Vrifions que tout fonctionne bien :
    • Lancer un client Web (Internet Explorer ou Netscape ou ...) et saisir cette URL http://127.0.0.1/. Le client Web affiche la page de EasyPHP et indique qu'il n'y a aucun rpertoire dans www.
    • Lancer un client Web (Internet Explorer ou Netscape ou ...) et saisir cette URL http://127.0.0.1/mysql/. Le client Web affiche la page de PhpMyAdmin.
    Si ces deux tests sont OK, nous pouvons passer l'tape suivante, l'installation de Talesta.
OU 1bis. Installer WampServer
  1. Rcuprer WampServer sur http://www.wampserver.com/
  2. Lancer wamp5_1.4.4.exe pour installer le tout dans le rpertoire de votre choix (dnomm par la suite dans ce document [RepEasyPHP])
  3. Une fois l'installation termine, lancer wampserver (dmarrer, programmes, Wampserver, start WampServer) si ce n'est pas fait avec la case a cocher lors de l'install. Une icne (genre compteur de vitesse) apparait dans la barre de tches, signifiant que wampServer est lanc).
  4. Vrifions que tout fonctionne bien :
    • Lancer un client Web (Internet Explorer ou Netscape ou ...) et saisir cette URL http://127.0.0.1/. Le client Web affiche la page de WampServer et indique qu'il n'y a pas de projets.
    • Lancer un client Web (Internet Explorer ou Netscape ou ...) et saisir cette URL http://127.0.0.1/phpmyadmin/. Le client Web affiche la page de PhpMyAdmin.
    Si ces deux tests sont OK, nous pouvons passer l'tape suivante, l'installation de Talesta.
2. Une fois easyphp ou WAMPSERVER install
  1. Copier le contenu du zip de talesta dans [RepEasyPHP]\www.
  2. Il vous faut crer une base de donnes pour stocker vos donnes. Le nom par dfaut de cette base est "talesta". Lancer un client Web (Internet Explorer ou Netscape ou ...) et saisir cette URL http://127.0.0.1/mysql/. Le client Web affiche la page de PhpMyAdmin. Taper le nom de la base sous l'intitul "Crer une base de donnes", puis appuyer sur le bouton "Crer". La base doit se crer et PHPMyAdmin doit s'tre plac dans cette base et indiquer qu'il n'existe aucune table dans cette base.
Mthode en utilisant le script d'Anthor
  1. Lancer un client Web (Internet Explorer ou Netscape ou ...) et saisir cette URL http://127.0.0.1/talesta4/. Le client Web vous redirige sur l'url http://localhost/talestaBD/main/install.php?action=newinstall
  2. Renseigner le formulaire en tenant compte des indications donnes. Les informations les plus souvent renseignes sont: et voici celui pour un compte FREE. (il faut juste changer www.talesta dans base de donnes et nom d'utilisateur avec votre compte ainsi que le mot de passe)
  3. Le script excute les requtes SQL pour crer les tables et les affiche. Chaque requte est suivie d'un OK ou d'un KO. La fin de l'affichage du script doit tre '208 instructions se sont executes correctement sur 208.' Cliquer sur 'Continuer' Seul souci rencontr une fois lors des tests (install sur une cle USB): Le temps d'excution des requtes SQL fait que le script PHP met plus de 30 sec s'excuter, ce qui gnre une erreur 'PHP Fatal error: Maximum execution time of 30 seconds exceeded in ....' Dans ce cas, il faut modifier dans le fichier php.ini le paramtre max_execution_time pour l'augmenter et redmarrer Easyphp ou wampServer et recommencer.
  4. Le script cre alors le fichier include/config.php qui recense les donnes saisies et affiche 'criture du fichier de configuration ../include/config.php... OK' et renomme le fichier install.php pour viter que quelqu'un puisse rejour le script (ce qui ferait perdre vos donnes)
