Nom du sort : Le nom du Sort, (64 char max)
Type de sort : Le Type de sort
Sous Type de sort : Le Sous type du Sort
Effets Mini sur la cible: Les Effets Minimum que va provoquer le sort sur la cible:
  • les dgts sur les points de vie (PV) si le sort est de type Attaque,
  • les dgts sur les points d'action (PA) si le sort est de type Paralysie,
  • les apports sur les points de vie (PV) si le sort est de type Soin
Effets Max sur la cible: Les effets Maximum que va provoquer le sort sur la cible (cf. Effets Min)
Prix du sort: Prix demandé pour apprendre le sort dans les magasins. Determine aussi le prix pour recharger un sort
Description: Description du sort (BREVE !!)
Place: La place que prendra le sort dans le grimoire du PJ
Image (touchez pas): Permet de surdefinir l'image d'un sort. Doit etre exprimé relativement au repertoire /images/. Par exemple, pour faire une boule de feu avec une image de livre, entrez Icones/IconLivre.png
Permanent: Determine si les sort est permanent. Un Sort permanent ne peut pas perdu en cas de mort.
Anonyme: Determine si le sort est Anonyme. Un sort anonyme permet une action sur quelqu'un sans qu'il connaisse l'identite du lanceur.
Charges (-1 = infini) : Le nombre de charges d'un Sort. -1 represente des charges infinies
Caracteristique : La caracteristique qui sera utilisée dans les jets de dés.
Competence : La competence qui sera utilisée dans les jets de dés.
