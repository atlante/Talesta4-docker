Nom de l'objet : Le nom de l'Objet, (64 char max)
Type d'objet : Le Type de l'objet
Degats Mini : Degats Minimum de l'objet. Si l'objet est une clef, degats mini represente l'id du lieu de depart. Si l'objet est une armure, degats mini represente le nombre de points de degats que l'armure absorbe. Si l'objet est un passe partout, degats mini represente le bonus minimum au crochetage que procure le passe. Enfin, si l'Objet est un livre, degats mini represente le gain minimum d'Xp que procurera sa lecture. Si l'objet est une nourriture, degats min represente le gain (ou perte) de PVs qu'entraine sa consommation.
Degats Max : les Degats Maximum que pourra faire l'objet. Si l'objet est une clef, degats max represente l'id du lieu d'arrivee. Si l'objet est un apsse partout, degats max represente le bonus maximum au crochetage que confere l'utilisation du passe. Si l'objet est un livre, degats max represente l'XP maximum que sa lecture peut rapporter. Si l'objet est une nourriture, degats max represente le gain (ou perte) de PAs qu'entraine sa consommation.
Durabilite (-1 = infini) : La durabilite de l'objet. -1 represente un objet indestructible.
Prix de l'objet: Le prix de base de l'objet lorsqu'on l'achete ou qu'on veut le revendre. Compte aussi pour calculer le cout d'une reparation ou d'une recharge de l'objet
Description: Une description (BREVE !!!) de l'objet
Poids: Le poids de l'objet. Sert a calculer si un PJ peut le proter ou pas.
Image (touchez pas): Permet de surdefinir l'image d'un objet. Doit etre exprimé relativemetn au repertoire /images/. Par exemple, pour faire une epee avec une image de livre, entrez Icones/IconLivre.png
Permanent: Determine si l'objet est permanent. Un objet permanent ne peut pas etre donné a un autre PJ et ne peut pas etre pris par quelqu'un qui fouille un cadavre.
Anonyme: Determine si l'objet est Anonyme. Un objet anonyme permet une attaque sur quelqu'un sans qu'il connaisse l'identite de l'aggresseur. Reservé uniquement aux armes.
Munitions (-1 = infini) : Les munitions de l'objet. -1 represente un objet aux munitions infinies. Il va de soit qu'une epee par exemple, doit avoir des munitions infinies. Modif Hixcks: Les doses pour une nourriture
Caracteristique : Si l'objet est une arme, caracteristique represente la carac qui va etre utilisé dans le jet de dé. Si l'objet est un livre, carac represente la caracteristique qui va beneficier de l'augmentation d'XP
Competence : Si l'objet est une arme, comp represente la competence prise en comtpe dans le lancé de dé. Si l'objet est une armure, competence represente quel type de degats va etre absorbé. Si l'objet est un livre ET que carac est vide, comp represente la competence qui va beneificier du gain d'XP
Competence Spé: Si l'objet est une arme, represente ses possibilites speciales. Vampire provoque des degats sur la cible ET soigne l'attaquant. Etourdissant retire des PAs a la cible plutot que des PVs
Utilisable uniquement parL'objet ne sera utilisable uniquement par un PJ ayant cet état temporaire. Comme les états temporaires incluent les races, l'age... On peut limiter l'utilisation d'un objet à une race...
