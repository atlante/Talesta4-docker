Lieu de Départ : Le point de depart du chemin
Lieu d'Arrivée : L'arrivée du chemin
Difficulte : La difficulté du chemin. Represente le lancé de dé du programme lors d'un lieu passage,d'un lieu aller, d'un lieu nage/escalade, le montant à payer pour un lieu péage
Type : Le type du chemin
  • Lieu Entrer = Deplacement gratuit
  • Lieu Passage = porte fermée a clef
  • Lieu Guilde = Necessite un mot de passe
  • Lieu Aller = Deplacement qui coute autant de PAs que la distance
  • Lieu Secret = Le joueur doit fouiller le lieu pour trouver le chemin
  • Lieu Escalade/Nage = A choisir en fonction de la carte du monde du jeu
Pass : Le mot de passe (pour le Lieu Guilde)
Distance : La distance (Cout en PAs pour le Lieu Aller)
Retour ?Creation automatique du chemin inverse ou pas. Si oui, le chemin inverse est cree, exactement identique. SAUF dans le cas d'un lieu guilde, ou le retour est un lieu entrer (besoin d'un pass pour entrer, pas pour sortir). Cette option est vivement conseillée.
Note : Lorsqu'un lieu passage est cree, une clef correspondante est inseree dans les objets. Lorsqu'un joueur ouvre une porte et souhaite la tenir ouverte, il faut imperativement que la base contienne une clef qui correspond a cette porte.
A noter également qu'une seule clef est créé, meme si le retour est coché, une clef pouvant toujours marcher dans les deux sens.
