Fichiers modifis pour l'intgration avec PHPBB

Il ne vous reste plus qu' faire de mme avec le forum de votre choix

Version 3.4

Le moteur est compatible dsormais avec PHPBB et Phorum5 Grace la classe forum. Chaque forum son propre fichier (dans include/forum) et reproduit les fonctionnalits de la classe forum. Le choix du fichier se fait dans include/http_get_post.php 
	if (defined('IN_FORUM') && (IN_FORUM==1)) {
		switch($typeforum)
		{
			case 'phpBB': 
				include( '../include/forum/forum_phpBB2.'.$phpExtJeu);
				break;

			case 'phorum':
				include( '../include/forum/forum_phorum5.'.$phpExtJeu);
				break;
			....
La variable $typeforum est modifie dans admin/configuration gnrale (admin/config.php), 
			$liste_type_forum=array(
				"0"=>"phpBB",
				"1"=>"phorum"
			);
Pour le moment, il faut aussi crer un fichier PostInstallXXXX.sql avec XXXX etant le type de forum, pour crer, essentiellement, les types de membres (PJ_PNJ et MJ), des forums, des topics et la configuration gnrale du forum. Note: Ces fichiers seront prochainement remplacees par des methodes de la classe forum pour que toute l'install soit scripte.
