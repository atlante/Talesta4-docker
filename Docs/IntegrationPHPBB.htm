Fichiers modifiés pour l'intégration avec PHPBB

Il ne vous reste plus qu'à faire de même avec le forum de votre choix

Version 3.4

Le moteur est compatible désormais avec PHPBB et Phorum5 Grace à la classe forum. Chaque forum à son propre fichier (dans include/forum) et reproduit les fonctionnalités de la classe forum. Le choix du fichier se fait dans include/http_get_post.php 
	if (defined('IN_FORUM') && (IN_FORUM==1)) {
		switch($typeforum)
		{
			case 'phpBB': 
				include( '../include/forum/forum_phpBB2.'.$phpExtJeu);
				break;

			case 'phorum':
				include( '../include/forum/forum_phorum5.'.$phpExtJeu);
				break;
			....
La variable $typeforum est modifiée dans admin/configuration générale (admin/config.php), 
			$liste_type_forum=array(
				"0"=>"phpBB",
				"1"=>"phorum"
			);
Pour le moment, il faut aussi créer un fichier PostInstallXXXX.sql avec XXXX etant le type de forum, pour créer, essentiellement, les types de membres (PJ_PNJ et MJ), des forums, des topics et la configuration générale du forum. Note: Ces fichiers seront prochainement remplacees par des methodes de la classe forum pour que toute l'install soit scriptée.
